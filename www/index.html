Bitcoin Bottlecaps

Mint your own Bottlecaps

Bitcoin Bottlecaps is an easy way to bring virtual value from Bitcoin into physical, tangible form. The project is open source.

If you find a Bitcoin Bottlecap, simply scan the QR code to be able to redeem it.

All minting and redeeming is done client-side, so there is minimal opportunity for foul play on the server-side.
